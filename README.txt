In the project file, using the terminal:

Initialize Node.js:    npm init -y
------------------------------------
Install Node Express:   npm install express.

------------------------------------
Install Express- Handlebars:   npm install express-handlebars

------------------------------------
Run Server:   node app.js


------------------------------------
------------------------------------
To test using Jest:   npm install jest supertest --save-dev

------------------------------------
run the jest test:   npm test
